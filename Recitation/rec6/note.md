## Find the set of PO allocations

.bg-washed-green.b--dark-green.ba.bw2.br3.shadow-2.ph2.mt2[

**the set of PO allocations**: the set of feasible allocations $\{(x_1, y_1), (x_2, y_2) \}$ at which no one can be make better off without someone else being worse off.  

]
